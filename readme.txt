== Installation ==

1. Download the extension file - 'woocommerce_bulk_2.6.zip'

2. Log in into your WordPress admin panel

3. Go to Plugins > Add New

4. Click Upload Plugin

5. Click Choose File

6. Select the extension file - 'woocommerce_bulk_2.6.zip'

7. Click Install Now

8. Enter your FTP credentials and click Proceed

9. Plugin is installed, click Activate Plugin

10. Now you successfully activated Ship200 Bulk plugin, click Settings

11. Enter your Ship200 secret key in the Ship200 Key field and click Update settings
